Bot Commands

User Commands

/start

Start the bot and see registration options for the current event

/state

Check your registration status and see available spots

Admin Commands

/addevent Name;YYYY-MM-DD;Capacity

Create a new event with name, date, and capacity limit

/qrcode

Generate a QR code for event check-in

/export

Download registration data as CSV file

/remove @username

Remove a user from all registrations

How It Works

1

Create Event

Admin creates an event with name, date, and capacity using the /addevent command

2

Users Register

Users send /start to the bot and tap the register button to sign up

3

Check-in

At the event, attendees scan the QR code to mark their attendance

4

Export Data

Admin exports registration and attendance data for records

Installation

# Clone the repository
git clone https://github.com/adubovskoy/meetupbot.git
cd meetupbot

# Configure the bot
cp .env.example .env
nano .env  # Add your BOT_TOKEN and ADMIN_USERS

# Build and run
go build
./meetupbot

# Alternative: pass config via command line
BOT_TOKEN=your_token ADMIN_USERS=admin1,admin2 ./meetupbot
